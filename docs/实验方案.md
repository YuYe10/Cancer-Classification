# 乳腺癌分子亚型分类实验方案（精简版）

## 1. 目标

在 TCGA-BRCA 数据上，比较四种分类方法在同一严格评估口径下的表现：

- RNA-only（单组学基线）
- Concat（早期融合）
- MOFA（潜表示融合）
- Stacking（晚期融合）

重点回答两个问题：

- 不同融合策略是否能稳定提升分类准确率。
- 提升来自哪里（信息互补、降噪表示、集成决策边界）。

## 2. 数据与标签

- RNA: datasets/TCGA-BRCA.star_counts.csv
- Methylation: datasets/TCGA-BRCA.methylation450.csv
- Clinical label: datasets/brca_pam50.csv

标签空间为 LumA、LumB、HER2、Basal。样本对齐后，对样本数不足的类别进行自动剔除并记录。

## 3. 统一实验协议

### 3.1 评估口径

采用 repeated stratified k-fold 作为主结论口径：

- folds = 5
- repeats = 3
- 总计 15 个测试折

### 3.2 防泄露约束

每个 fold 内严格遵循：

1. 先划分训练折/测试折
2. 仅用训练折做特征筛选与标准化拟合
3. 用训练折拟合融合模块
4. 仅在测试折做独立推断与评估

MOFA 分支采用“训练折拟合 + 测试折投影”的评估路径，不再将测试样本混入训练拟合。

### 3.3 指标

主指标：

- Accuracy
- Balanced Accuracy
- Macro-F1

统计汇总：

- mean ± std
- 95% CI
- 分类报告与混淆矩阵

### 3.4 证据链顺序

为了让实验改进的逻辑链更清晰，正文与脚本统一按下面顺序组织证据：

1. 先用 RNA-only 作为单模态基线，给出最低复杂度参照；
2. 再用 Concat 检验跨组学早期融合是否带来稳定增益；
3. 然后用 MOFA 检查低维潜表示能否进一步降噪；
4. 最后用 Stacking 评估更高复杂度的决策级融合上限；
5. 在主结果之外，追加稳定性、类别级误差和 SHAP 解释，说明“为什么会这样”。

## 4. 方法对比与预期差异

### 4.1 RNA-only

- 作用：构建可比较的最低复杂度基线。
- 预期：在样本量较小与类别不平衡下，性能受限但稳定。

### 4.2 Concat

- 作用：直接拼接 RNA + 甲基化特征。
- 预期：利用跨组学互补信息，准确率高于 RNA-only。

### 4.3 MOFA

- 作用：先学低维潜因子，再做分类。
- 预期：通过潜表示降低噪声与冗余，优于简单拼接。

### 4.4 Stacking

- 作用：模态内基模型 + 元分类器晚期融合。
- 预期：作为融合复杂度更高的对照方法，若样本规模与超参数足够合适，可能进一步优化决策边界；在当前小样本条件下更重要的是验证它是否真的优于 Concat，而不是默认它一定最高。

## 8. 详细继续改进方案（围绕“为什么做、做了什么、效果如何”）

### 8.1 为什么做此研究（报告必写要点）

1. BRCA 亚型具有明显临床异质性，直接影响治疗路径与预后判断；
2. 单组学模型存在上限，多组学融合可能带来互补信息；
3. 课程作业不仅考察模型分数，更考察“问题定义-方法比较-结果解释-边界说明”的完整研究能力。

### 8.2 做了什么实验（当前已完成）

1. 主线四方法统一对比：RNA-only / Concat / MOFA / Stacking；
2. 统一 repeated stratified CV（5x3）评估；
3. 三类核心指标：Accuracy、Balanced Accuracy、Macro-F1；
4. 统计检验：95\% CI + 置换检验；
5. 类别级分析：LumA/LumB/Basal 召回率与难分类别识别。
6. 图表闭环：主结果图、稳定性图、SHAP summary 和类别表都由脚本自动生成并写入论文；其余诊断图统一归档。

### 8.3 达到了什么效果（当前证据）

1. Concat 在三项主指标上取得最优均值（Accuracy 0.9012，Balanced Accuracy 0.9074，Macro-F1 0.8999）；
2. 类别级层面，LumB 是所有方法的共同难点，Concat 对该类召回率最高；
3. 显著性检验尚未达到确定性优势，当前结论应写为“趋势性优势 + 统计边界透明”。

### 8.4 后续改进执行清单（高分导向）

1. \textbf{短期}：增加重采样轮次、补全多指标显著性图表、完善类别级错误分析；
2. \textbf{中期}：引入外部验证集，验证主线结论可迁移性；
3. \textbf{长期}：补充生物学机制映射（通路/文献证据）与更强融合模型探索；
4. \textbf{归档}：每轮迭代结束后先更新白名单图，再清理无用或明显错误图片，避免论文材料集继续膨胀。
