# 乳腺癌分子亚型分类实验方案（精简版）

## 1. 目标

在 TCGA-BRCA 数据上，比较四种分类方法在同一严格评估口径下的表现：

- RNA-only（单组学基线）
- Concat（早期融合）
- MOFA（潜表示融合）
- Stacking（晚期融合）

重点回答两个问题：

- 不同融合策略是否能稳定提升分类准确率。
- 提升来自哪里（信息互补、降噪表示、集成决策边界）。

## 2. 数据与标签

- RNA: datasets/TCGA-BRCA.star_counts.csv
- Methylation: datasets/TCGA-BRCA.methylation450.csv
- Clinical label: datasets/brca_pam50.csv

标签空间为 LumA、LumB、HER2、Basal。样本对齐后，对样本数不足的类别进行自动剔除并记录。

## 3. 统一实验协议

### 3.1 评估口径

采用 repeated stratified k-fold 作为主结论口径：

- folds = 5
- repeats = 3
- 总计 15 个测试折

### 3.2 防泄露约束

每个 fold 内严格遵循：

1. 先划分训练折/测试折
2. 仅用训练折做特征筛选与标准化拟合
3. 用训练折拟合融合模块
4. 仅在测试折做独立推断与评估

MOFA 分支采用“训练折拟合 + 测试折投影”的评估路径，不再将测试样本混入训练拟合。

### 3.3 指标

主指标：

- Accuracy
- Balanced Accuracy
- Macro-F1

统计汇总：

- mean ± std
- 95% CI
- 分类报告与混淆矩阵

## 4. 方法对比与预期差异

### 4.1 RNA-only

- 作用：构建可比较的最低复杂度基线。
- 预期：在样本量较小与类别不平衡下，性能受限但稳定。

### 4.2 Concat

- 作用：直接拼接 RNA + 甲基化特征。
- 预期：利用跨组学互补信息，准确率高于 RNA-only。

### 4.3 MOFA

- 作用：先学低维潜因子，再做分类。
- 预期：通过潜表示降低噪声与冗余，优于简单拼接。

### 4.4 Stacking

- 作用：模态内基模型 + 元分类器晚期融合。
- 预期：在不增加数据泄露风险的前提下，进一步提升泛化能力。

## 5. 执行命令

主线一键运行：

```bash
bash run_all.sh
```

严格 CV 批处理：

```bash
bash scripts/run_strict_cv.sh
```

稳健评估与统计比较：

```bash
bash scripts/run_robust_evaluation.sh
```
