\begin{abstract}
我们围绕课程任务"单癌种（BRCA）内部的分子亚型分类"完成了可复现的多组学实验。基于 TCGA-BRCA 的 RNA-seq、DNA 甲基化与 PAM50 标签，我们在统一 repeated stratified CV（5 folds $\times$ 10 repeats）口径下比较了 RNA-only、Concat、MOFA 和 Stacking 四条主线。为保证公平性，特征筛选、标准化、MOFA 拟合与 Stacking 元特征构造均严格限定在训练折内，测试折仅用于独立推断。

在当前数据对齐结果下，HER2 类样本数不足（$<2$）被自动剔除。主结果显示：Concat 的 Accuracy 最优（0.9012，95\% CI [0.8609, 0.9415]），RNA-only（0.8655）与 Stacking（0.8488）次之，MOFA（0.7857）波动较大。置换检验显示组间差异尚未达到显著水平，说明在小样本条件下应优先解读“稳定趋势 + 置信区间”而非单一高分。整体上，我们完成了课程要求的数据整合、算法比较、分类评估与研究报告闭环，并明确了后续提升路径。

\vspace{0.5em}
\noindent\textbf{关键词：}多组学融合；乳腺癌；TCGA-BRCA；分子亚型分类；MOFA；Stacking；严格交叉验证
\end{abstract}
