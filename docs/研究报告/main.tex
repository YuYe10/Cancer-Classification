\begin{abstract}
乳腺癌分子亚型分类对临床治疗决策具有重要意义。相比单组学建模，多组学融合可同时利用基因表达与表观遗传信息，提升亚型识别能力。本文基于 TCGA-BRCA 数据构建了统一、可复现的分类流程，在严格重复交叉验证口径下比较了四类方法：RNA-only、Concat、MOFA 与 Stacking。为保证评估公平性，所有特征选择、标准化与融合步骤均在训练折内完成，测试折仅用于独立评估。

本文重点关注两类问题：一是不同融合策略在准确率与稳定性上的差异；二是性能提升背后的方法学原因。实验显示，随着融合方式从单组学到早期融合、再到潜表示融合与晚期融合，整体分类性能呈逐步提升趋势。结果证明，多组学信息互补、潜在因子降噪以及分支级集成是提升分子亚型分类准确率的关键机制。

\vspace{0.5em}
\noindent\textbf{关键词：}多组学融合；乳腺癌；TCGA-BRCA；分子亚型分类；MOFA；Stacking；严格交叉验证
\end{abstract}
