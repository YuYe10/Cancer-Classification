\section{讨论}

\subsection{为什么 Concat 在当前结果中表现较好}

当前实验中 Concat 获得了较优结果，这并不令人意外。对于样本规模有限、组学数量较少、且各模态都经过高方差筛选的场景来说，简单拼接往往已经能够有效聚合信息，且不会引入过多额外训练不稳定因素。换句话说，当每个模态都已经被压缩到较合理的维度时，Concat 作为直接融合策略可以提供非常强的基线性能。

这类现象在多组学研究中很常见：更复杂的方法未必总能立即超过简单拼接，尤其是在样本数不足或超参数尚未充分优化的情况下。因此，本研究中的 Concat 结果应被视为一个重要参照，而不是“简单方法一定更差”的反例。

\subsection{MOFA 的优势与局限}

MOFA 的理论优势在于它并不直接把所有特征视作同等重要，而是通过潜在因子将跨组学共享信号提取出来。对于具有明显共享调控模式的疾病分类任务，这种机制往往能提供比拼接更干净的表示。然而，MOFA 的效果高度依赖训练设置：包括因子数、初始化方式、样本量与视图数等。

在当前实验中，MOFA 能够成功训练并产生低维表示，这一事实本身就说明其工程可用性较强。但从当前一次结果看，它尚未大幅超越 Concat，这提示我们需要从三个方面进一步分析：第一，因子数是否合适；第二，训练样本是否足够；第三，合并样本拟合一次的工程做法是否足以代表严格的泛化评估。论文中应对这些问题保持诚实，而不是只报告一个漂亮的结果。

\subsection{消融实验的意义}

消融实验的核心价值，在于把“结果好不好”变成“为什么好”。去甲基化与去 RNA 的实验说明，单一组学都能提供一定信息，但其分类边界不如多组学联合输入稳定；不做特征选择的实验则说明，高维噪声会显著损害模型性能。这两点合在一起，实际上证明了本文方法链条中的两个关键设计：多模态融合与特征筛选都不是装饰，而是任务成功的必要环节。

这类结论对于课程报告尤其重要，因为它不仅说明实验做出来了，还说明“为什么要这么做”。对于老师或评审而言，这比单纯展示一个准确率更有说服力。

\subsection{当前研究的局限性}

尽管当前实验已经形成完整链条，但仍然存在几项局限。首先，样本量仍然偏小，单次随机划分容易造成指标波动；其次，当前实现的 MOFA 分支仍有工程约束，严格的训练/测试独立变换流程尚未完全打通；再次，实验目前主要聚焦分类性能，而对生物学解释的展开还不够充分。

此外，本文尚未引入更强的统计验证，例如多随机种子重复实验、显著性检验或置信区间估计。对于正式论文来说，这些内容是增强结论可信度的重要补充。后续若要进一步写成更完整的研究论文，建议将这些部分作为重点扩展。

\subsection{未来可扩展的方向}

基于现有框架，后续扩展空间非常明确：
\begin{enumerate}
  \item 引入更多组学视图，例如 CNV 或突变负荷，以增强跨模态信息量；
  \item 将分类器替换为随机森林、XGBoost 或线性模型，对比不同判别器的表现；
  \item 加入 SHAP 或基于 MOFA 因子的特征解释分析，形成“性能 + 可解释性”双重输出；
  \item 补充生存分析任务，将当前亚型分类框架进一步延伸到预后预测。
\end{enumerate}

这些方向并不意味着必须同时实现，而是说明当前报告并不是终点，而是一个可以不断向外扩展的研究平台。
