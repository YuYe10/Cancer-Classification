\section{讨论}

\subsection{方法差异的解释}

四种方法形成由浅入深的融合链条：RNA-only 提供单组学基线，Concat 引入跨模态互补，MOFA 在潜空间压缩冗余，Stacking 在决策层进一步融合。若结果呈现递进提升，通常意味着多组学互补信息被逐步有效利用。

\subsection{为什么严格 CV 是主结论口径}

在小样本高维任务中，单次切分结果方差较大，容易出现方法同分或排序反转。repeated CV 通过多次分层重采样提供更稳健的性能估计，因此更适合作为论文主证据。

\subsection{MOFA 的价值与边界}

MOFA 的核心价值在于潜表示建模。本文实现已将 MOFA 纳入训练折拟合与测试折投影流程，提升了评估一致性。但其性能仍受样本规模、因子数与分类器配置影响，需要结合统计区间而非单点分数解读。

\subsection{当前局限}

本文仍存在三点局限：

\begin{enumerate}
    \item 样本规模有限，部分类别样本稀少；
    \item 外部队列验证尚未完成；
    \item 超参数搜索空间仍可扩展。
\end{enumerate}

\subsection{后续工作}

下一步将沿两条路径推进：

\begin{enumerate}
    \item 扩展稳健性证据：更多随机种子、外部验证与显著性检验；
    \item 面向性能目标调参：优先优化 Stacking 与 MOFA 相关配置。
\end{enumerate}
