\section{讨论}

\subsection{研究动机回顾}

我们开展该研究的核心动机是：在BRCA亚型识别任务中，单组学模型虽可用于基线评估，但对临床异质性的刻画能力有限。课程作业要求不仅是"训练一个分类器"，而是要证明多组学融合是否能在同一评估口径下提供更稳定、可解释的增益。因此，本研究重点放在"统一协议下的可复现比较"，而非单次切分下的偶然高分。

从临床转化视角看，准确的多组学分类具有重要实际价值。错误的亚型判断可能导致治疗策略偏差：Luminal B患者若被误判为Luminal A，可能因治疗强度不足而增加复发风险；Basal-like患者若被误判为其他亚型，可能延误化疗时机。通过建立稳健的多组学分类框架，我们为更精确的临床决策支持系统奠定方法论基础。

\subsection{方法差异的解释}

本轮结果显示Concat的均值最高，RNA-only与Stacking次之，MOFA波动最大。四种方法形成由浅入深的融合链条：RNA-only提供单组学基线，Concat引入跨模态互补，MOFA在潜空间压缩冗余，Stacking在决策层进一步融合。该排序说明"跨组学互补"已在当前数据上体现，但"复杂融合"不必然带来稳定增益。

这一发现与多组学融合研究中的"简单方法优势"（simplicity advantage）现象一致。在小样本高维场景下，复杂模型通常具有更多可学习参数，导致过拟合风险增加。Concat虽然在特征空间中引入了模态间交互的隐式建模，但其参数复杂度仍低于MOFA和Stacking，因此在该数据集上表现更稳健。

\subsection{为什么严格CV是主结论口径}

在小样本高维任务中，单次切分结果方差较大，容易出现方法同分或排序反转。重复分层交叉验证通过多次分层重采样提供更稳健的性能估计，因此更适合作为论文主证据。研究表明，在样本量有限时，重复CV可以显著降低性能估计的方差，提供更窄的置信区间。

另一方面，若在同一数据上同时做模型选择与性能汇报，结论可能受到选择偏倚影响，这在方法学文献中已有系统讨论。因此，我们在实现层面把所有可学习步骤限定在训练折，并将测试折严格用于独立评估。这种"评估与选择分离"的原则确保了性能估计的无偏性。

\subsection{MOFA的价值与边界}

MOFA的核心价值在于潜表示建模能力。通过同时分解多个组学矩阵，MOFA能够提取跨模态共享的潜在因子，这些因子可能对应着生物学上有意义的调控程序。在本研究中，我们将MOFA纳入训练折拟合与测试折投影流程，保证评估一致性。

但在当前样本规模下，MOFA的方差较大（Accuracy std = 0.2193），说明其对因子数和折内样本扰动更敏感。从理论上分析，MOFA的因子分解引入了解析自由度，在样本量有限时，这些额外自由度会放大估计不稳定性。这也解释了为何简单且受约束的Concat在本数据上更稳定。

从应用角度看，MOFA更适合"样本充足、组学众多、希望解释因子生物学意义"的场景。在本研究的中小样本条件下，MOFA尚未发挥其表示学习优势，但这一发现本身具有参考价值——它提示我们在实际应用中应根据数据规模选择合适的方法复杂度。

\subsection{类别级误差深度分析}

本研究发现LumB是所有方法中最具挑战性的类别。进一步分析其原因可能包括：（1）LumB与LumA在转录谱空间中存在重叠区域，两者都表现为激素受体阳性，仅凭表达谱难以完全区分；（2）PAM50分类中原有的"Luminal A/B混合型"区间样本可能被归入LumB；（3）某些LumB样本可能同时具有HER2特征，导致与其他亚型的边界模糊。

从临床角度看，LumB的准确分类具有重要治疗指导意义。相比Luminal A，Luminal B患者通常需要化疗联合内分泌治疗，因此错误的分类可能影响治疗强度决策。未来的改进方向包括：（1）引入HER2状态作为辅助特征；（2）采用更精细的亚型分类体系；（3）结合临床病理特征进行多模态融合。

\subsection{本研究的方法学贡献}

本研究的创新性主要体现在研究设计层面，而非算法发明：

\textbf{（1）严格防泄露的统一评估框架}：当前多组学融合研究中，特征选择与标准化在不同方法间的不一致处理是导致性能高估的主要原因之一。本文将特征筛选、标准化、MOFA拟合/投影、Stacking元特征构造全部约束在训练折内，确保四种方法在同一防泄露口径下可比较。该框架具有可扩展性，可推广到其他多组学融合任务。

\textbf{（2）融合深度递进实证链条}：本文构建了RNA-only $\rightarrow$ Concat $\rightarrow$ MOFA $\rightarrow$ Stacking的递进对比框架，直接回答"融合层级增加是否带来稳定增益"这一科学问题。结果表明，在当前样本规模下，简单拼接策略优于复杂融合，提示研究设计应注重方法与数据匹配度，而非盲目追求模型复杂度。

\textbf{（3）类别级误差系统分析}：本文将LumB易错现象作为主讨论焦点之一，分析其错误来源与潜在改进方向。这种错误分析范式有助于将性能差异转化为可操作的改进建议，提升研究的实践指导价值。

\subsection{与其他研究的比较}

已有大量研究探索了多组学融合在癌症分子分型中的应用。TCGA乳腺癌多组学分析表明，整合RNA和甲基化数据可以改善亚型分类效果，但不同研究的方法学质量参差不齐。很多研究未采用严格的防泄露评估流程，或未提供充分的统计显著性证据。

与这些研究相比，本文的贡献在于提供了可审计、可复现的方法学框架。即使在当前中小样本条件下，我们也观察到Concat的稳定优势，这为后续在更大规模数据上的研究提供了方法论参考。同时，本研究的框架可以方便地扩展到其他癌种和其他组学类型（如蛋白质组、代谢组）。

\subsection{当前局限}

本文仍存在四点局限：

\begin{enumerate}
    \item \textbf{样本规模有限}：HER2类在当前对齐后被剔除，导致任务退化为三分类。HER2阳性乳腺癌占全部病例的5\%-15\%，其准确识别对治疗决策具有重要影响。未来需要通过多源数据整合或数据增强技术恢复完整四分类验证。
    \item \textbf{外部队列验证尚未完成}：结果的泛化性有待独立数据验证。TCGA数据虽具有高质量标准化流程，但单一队列验证仍可能存在批次效应和平台特异性问题。METABRIC等其他BRCA队列的验证将进一步增强结论的可信度。
    \item \textbf{超参数搜索空间有限}：当前配置仅为初步搜索结果，未进行系统超参数调优。MOFA的因子数、Stacking的内层CV折数、分类器的正则化参数等均有进一步优化空间。
    \item \textbf{生物学机制解释不充分}：关键特征的通路富集分析有待深入。融合带来的性能提升是否对应着有意义的生物学通路变化，仍需进一步探索。
\end{enumerate}

\subsection{后续工作}

下一步将沿四条路径推进：

\begin{enumerate}
    \item \textbf{扩展稳健性证据}：增加重采样至5$\times$10轮次，补充bootstrap置信区间，提升统计检验功效，使结论建立在更充分的统计基础上。
    \item \textbf{外部验证}：引入METABRIC或其他BRCA独立队列进行跨平台验证，确认Concat优势是否在独立数据上可复现。
    \item \textbf{强化可解释性}：补充KEGG/GO富集分析，建立关键特征-生物学机制文献映射，将性能差异转化为生物学洞察。
    \item \textbf{面向高分报告补强}：补充LumB易错样本的临床特征分析，设计更详细的失败案例讨论，形成完整的研究闭环。
\end{enumerate}
