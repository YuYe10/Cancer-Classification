\section{写作与扩写说明}

当前版本的 LaTeX 源码已经具备完整的论文结构，可以直接作为中文研究报告的初稿使用。由于用户希望最终篇幅约为 5 万字，因此建议采用“分段扩写”的方式继续完善，而不是一次性在单个文件中堆叠过长文本。这样更便于后续修改、审阅和插入图表。

\subsection{建议的扩写顺序}

\begin{enumerate}
  \item 先补充相关工作综述，扩展到多组学分类方法谱系、MOFA 与图神经网络等主题；
  \item 再扩写方法章节中的数学形式，对特征筛选、标准化、SVM 与 MOFA 进行更系统的公式化描述；
  \item 随后补充结果章节中的图表分析，包括混淆矩阵、ROC 曲线、t-SNE 与消融对比表；
  \item 最后扩展讨论与总结，联系临床意义、可解释性、局限性与未来方向。
\end{enumerate}

\subsection{与当前仓库实现的对应关系}

本文中的所有叙述都尽量与当前仓库实现保持一致，尤其是以下几点：
\begin{itemize}
  \item 主实验由 \texttt{run\_all.sh} 统一调度；
  \item 消融实验由 \texttt{scripts/run\_ablation.sh} 统一调度；
  \item 数据读取、样本对齐、标签映射与预处理顺序已在代码中固定；
  \item MOFA 分支已经兼容当前 \texttt{mofapy2} 接口，但其使用方式需要在论文中如实说明。
\end{itemize}

\subsection{后续可继续加入的内容}

如果需要继续将本文扩展到更接近正式论文或课程大作业的篇幅，可以按以下方向补充：
\begin{itemize}
  \item 增加文献引用，尤其是 TCGA、UCSC Xena、PAM50、MOFA 与多组学分类相关文献；
  \item 插入实验流程图、数据处理流程图、主实验对比图与消融结果图；
  \item 加入“实验环境与复现说明”小节，说明 Python、依赖包和运行命令；
  \item 加入“附录”章节，展示配置文件示例、关键代码片段与结果日志。
\end{itemize}

\subsection{当前版本定位}

当前稿件适合被视为“研究报告初稿 + 论文骨架 + 可复现实验说明”的结合体。它不是最终定稿，但已经满足从零开始搭建中文论文 LaTeX 工程的基本要求。后续若继续分段执行，我可以在此基础上继续扩展：相关工作、数学推导、实验表格、图表占位与参考文献，逐步向完整的五万字版本逼近。
