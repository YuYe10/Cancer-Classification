\section{结论}

我们围绕 BRCA 分子亚型分类任务，完成了统一、可复现的多组学实验流程，并在严格 repeated CV 口径下比较了 RNA-only、Concat、MOFA 与 Stacking 四种方法。该研究的直接目标是回答：多组学融合是否在课程任务场景下带来稳定、可解释的性能收益。

本研究的主要结论如下：

\begin{enumerate}
    \item 在本轮 BRCA 数据上，Concat 取得最高均值性能（Accuracy 0.9012，Balanced Accuracy 0.9074，Macro-F1 0.8999）；
    \item 置换检验未显示显著差异，提示当前结论应定位为“趋势性优势”而非统计学确定优势；
    \item 训练折内拟合、测试折独立推断的防泄露设计是保证结果可信度的关键。
\end{enumerate}

总体上，我们已完成课程任务要求的“数据整合-算法比较-指标评估-研究报告”闭环。本文的 novelty 主要体现为：在单癌种 BRCA 小样本条件下，建立了严格防泄露的统一评估框架，并以可复现方式给出融合递进链条的实证证据。

面向高分报告，下一步重点是三项补强：
\begin{enumerate}
    \item 提升统计功效：增加重采样轮次，检验当前趋势是否稳定收敛；
    \item 强化类别级解释：围绕 LumB 易错现象补充错分成因与关键特征分析；
    \item 完善机制证据：补充关键特征与文献机制映射，形成“结果-机制-边界-展望”完整闭环。
\end{enumerate}
