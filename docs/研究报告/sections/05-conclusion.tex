\section{结论}

本文围绕乳腺癌分子亚型分类任务，构建了统一、可复现的多组学实验流程，并在严格 repeated CV 口径下比较了 RNA-only、Concat、MOFA 与 Stacking 四种方法。

本文的主要结论如下：

\begin{enumerate}
    \item 统一评估协议能够有效减少方法比较中的口径偏差；
    \item 从单组学到多层融合的路线在整体上体现出可解释的性能递进；
    \item 训练折拟合与测试折独立推断是保证多组学评估可信度的关键。
\end{enumerate}

总体上，本文完成了从工程实现到论文主线证据的闭环。未来工作将围绕外部验证与系统调参继续提升模型稳定性与准确率上限。
