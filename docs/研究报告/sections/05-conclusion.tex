\section{结论}

我们围绕BRCA分子亚型分类任务，完成了统一、可复现的多组学实验流程，并在严格repeated CV口径下比较了RNA-only、Concat、MOFA与Stacking四种方法。该研究的直接目标是回答：多组学融合是否在课程任务场景下带来稳定、可解释的性能收益。

本研究的主要结论如下：

\begin{enumerate}
    \item \textbf{Concat取得最高均值性能}：在本轮BRCA数据上，Concat方法在Accuracy（0.9012）、Balanced Accuracy（0.9074）和Macro-F1（0.8999）三项指标上均表现最优，显著优于RNA-only基线。
    \item \textbf{统计显著性尚不充分}：置换检验未显示显著差异，提示当前结论应定位为"趋势性优势"而非统计学确定优势。这一发现符合小样本研究的普遍规律，需要更大规模数据或外部验证队列确认。
    \item \textbf{防泄露设计是保证可信度的关键}：训练折内拟合、测试折独立推断的严格流程确保了结果的可信度，为后续研究和临床应用奠定了方法论基础。
    \item \textbf{方法复杂度与数据规模需匹配}：简单Concat方法优于复杂MOFA/Stacking的现象提示，在中小样本条件下，应优先选择参数复杂度与数据量匹配的方法。
\end{enumerate}

\subsection{主要贡献}

本文的贡献主要体现在以下三个方面：

\textbf{研究设计创新}：在单癌种BRCA小样本条件下，我们建立了严格防泄露的统一评估框架。该框架将特征筛选、标准化、MOFA拟合与Stacking元特征构造全部约束在训练折内，确保四种方法在同一公平口径下比较。该框架具有可推广性，可应用于其他多组学融合任务的方法学评估。

\textbf{融合链条实证}：以可复现方式给出RNA-only $\rightarrow$ Concat $\rightarrow$ MOFA $\rightarrow$ Stacking的递进对比实证。结果表明，跨组学互补信息在当前数据上已体现，但复杂融合不必然带来稳定增益。这一发现对实际应用中选择合适的多组学融合策略具有指导意义。

\textbf{错误分析闭环}：将类别级误差（尤其LumB易错现象）纳入主讨论，形成面向后续改进的具体问题清单。错误来源的分类讨论有助于提升方法复现成功率，为临床转化应用中的误差控制提供参考。

\subsection{理论价值与实践意义}

从理论价值看，本研究为多组学融合研究中的方法选择提供了实证依据。已有文献多报道复杂融合方法的优势，但这些结论往往基于单一数据集或缺乏严格的防泄露评估。本研究通过系统性的方法比较，揭示了在小样本条件下简单方法的稳健性优势，为方法学讨论提供了新的证据。

从实践意义看，本研究为临床决策支持系统的开发提供了方法论基础。准确的分子亚型分类直接关系治疗策略选择：通过建立稳健的多组学分类框架，我们为更精确的临床决策支持系统奠定基础。特别是在资源受限的场景下，简单高效的Concat方法可能更具实际应用价值。

\subsection{创新性边界声明}

为避免过度主张，本文明确将novelty限定为"研究设计创新"而非"算法发明"：

\begin{enumerate}
    \item 在BRCA单癌种任务中实现了可审计的严格防泄露评估流程，为多组学融合研究的方法学质量树立标杆；
    \item 构建并实证比较了四层融合递进链条，给出可复算的性能趋势与统计边界，为方法选择提供量化依据；
    \item 将类别级误差纳入主讨论，形成面向后续改进的具体问题清单，提升研究的实践指导价值。
\end{enumerate}

\subsection{未来展望}

面向进一步研究，下一步重点包括：

\begin{enumerate}
    \item \textbf{提升统计功效}：增加重采样轮次至5$\times$10，检验当前趋势是否稳定收敛；同时引入bootstrap置信区间估计，提供更精确的不确定性量化；
    \item \textbf{外部验证}：引入METABRIC或其他BRCA独立队列进行跨平台验证，确认Concat优势是否在独立数据上可复现，增强结论的泛化可信度；
    \item \textbf{强化生物学解释}：补充关键特征与KEGG/GO富集分析，建立特征-机制文献映射，将性能差异转化为有意义的生物学洞察；
    \item \textbf{完善失败案例讨论}：深入分析LumB易错样本的临床特征与分子特征，形成完整的研究闭环，为临床转化应用提供更全面的参考。
\end{enumerate}
