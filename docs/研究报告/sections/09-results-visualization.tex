\section{结果与可视化分析}

\subsection{主实验结果汇总（严格 CV）}

本文主结果统一采用 repeated stratified CV（5 folds $\times$ 10 repeats）口径，共 50 个测试折。四种方法在同一评估协议下比较 Accuracy、Balanced Accuracy 与 Macro-F1，并报告均值与 95\% CI。所有主线图表均由脚本从真实日志自动生成。

\subsection{性能差异解释}

从方法机制看，性能提升可按以下链条理解：

\begin{enumerate}
    \item \textbf{RNA-only} 提供单组学可分性基线；
    \item \textbf{Concat} 利用 RNA 与甲基化互补信息，通常优于单模态；
    \item \textbf{MOFA} 通过潜因子压缩噪声与冗余，在小样本高维场景下可能有提升或不稳定性；
    \item \textbf{Stacking} 在分支层建模后再做元融合，进一步优化决策边界与泛化表现。
\end{enumerate}

结合表~\ref{tab:permtest-all} 可见，当前四方法的差异尚未达到统计显著。我们将其解释为：在小样本任务中，方法排序可作为“方向性证据”，但仍需更大样本或外部队列验证。

从主线排序看，Concat 在三项核心指标上均为第一，这与“RNA 表达与甲基化调控具有互补信息”的机制预期一致；同时，Stacking 并未稳定超过 Concat，提示在当前样本规模下，增加融合层级并不一定转化为稳定增益。

\subsection{类别级误差分析}

为回答“提升来自哪里”，我们进一步统计各方法在三类有效标签（LumA/LumB/Basal）上的 fold-level 平均召回率与 F1 分数。结果显示四种方法的最难类别均为 LumB，其中 Concat 的 LumB 召回率最高（0.846），相较 RNA-only（0.802）提升明显；MOFA 在该类别上的表现波动最大，与其整体方差较大现象一致。从 F1 分数看，LumA 与 Basal 类的表现相对稳定，而 LumB 的类内差异最大，提示该类别仍有显著的分类难度。

\subsection{小结}

正文结果部分聚焦一个核心问题：在统一口径下，四种方法是否呈现可复现的性能趋势。我们将“最优均值（Concat）+ 统计不显著 + 置信区间重叠”一并报告，避免把单一高分误写为确定性优势。

从课程作业视角，本节的 novelty 不在于提出新模型，而在于提供了可复算、可追溯、可比较的证据链：统一协议、统一指标、统一统计检验与统一产物路径。
