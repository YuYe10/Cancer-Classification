\section{结果与可视化分析}

\subsection{主实验结果汇总（严格 CV）}

本文主结果统一采用 repeated stratified CV（5 folds $\times$ 10 repeats）口径，共 50 个测试折。六种方法在同一评估协议下比较 Accuracy、Balanced Accuracy 与 Macro-F1，并报告均值与 95\% CI。所有主线图表均由脚本从真实日志自动生成。

基于 \texttt{outputs/logs/summary1/summary.csv} 中的 41 组实验结果，核心发现如下：

\begin{enumerate}
    \item \textbf{Concat-SVM 取得最优综合性能}：Accuracy=90.64\%，Macro-F1=90.03\%，Balanced Acc=90.56\%
    \item \textbf{MOFA 表现优异且稳定}：Accuracy=90.00\%，Macro-F1=89.03\%，与 Concat-SVM 差距$<$1\%
    \item \textbf{RNA 单模态已具备强分类能力}：Accuracy=89.78\%，仅比最优低 0.86 个百分点
    \item \textbf{Meth-only 性能明显偏低}：Accuracy=83.67\%，较 RNA 低约 6 个百分点
    \item \textbf{Stacking 未带来预期增益}：XGB+Logistic Stacking 仅达 85.56\%，低于所有单层融合方法
\end{enumerate}

\subsection{性能差异解释}

从方法机制看，性能排序可按以下链条理解：

\begin{enumerate}
    \item \textbf{RNA-only} 提供单组学可分性基线，已达到近 90\% 准确率，表明转录组数据包含丰富的亚型判别信息；
    \item \textbf{Concat-SVM} 利用 RNA 与甲基化互补信息，在 SVM 线性分类器下取得最高均值；
    \item \textbf{MOFA} 通过潜因子压缩噪声与冗余，在 20 因子配置下表现接近最优，说明降维后仍保留了关键判别特征；
    \item \textbf{Stacking} 在分支层建模后再做元融合，但 XGB+Logistic 配置下反而出现性能下降，提示在小样本场景下元学习器可能过拟合。
\end{enumerate}

结合表~\ref{tab:results-main} 可见，Concat、MOFA 与 RNA-only 三者置信区间高度重叠（差距$<$2.3\%），而 Stacking 的置信区间明显偏下。我们据此将结论定位为"方向性趋势"而非统计学确定优势。

从主线排序看，Concat-SVM 在三项核心指标上均为第一，这与"RNA 表达与甲基化调控具有互补信息"的机制预期一致；同时，Stacking 并未稳定超过简单融合方法，提示在当前样本规模下（$n\approx$150），增加融合层级并不一定转化为稳定增益。

\subsection{小结}

正文结果部分聚焦一个核心问题：在统一口径下，六种融合策略是否呈现可复现的性能趋势。我们将以下发现一并报告：

\begin{enumerate}
    \item \textbf{简单方法不劣于复杂方法}：Concat-SVM 以最简架构取得最优均值；
    \item \textbf{RNA 信息量充足}：单模态 RNA 已达到 89.78\% 准确率；
    \item \textbf{Stacking 存在过拟合风险}：所有 Stacking 变体均低于 Concat/MOFA/RNA baseline；
    \item \textbf{Methylation 是重要补充而非独立来源}：Meth-only 较弱，但对 Concat 有正向贡献。
\end{enumerate}

从课程作业视角，本节的 novelty 不在于提出新模型，而在于提供了可复算、可追溯、可比较的证据链：统一协议、统一指标、统一统计检验与统一产物路径。
