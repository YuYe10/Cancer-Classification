\section{误差分析与敏感性讨论}

\subsection{类别级别的识别差异}

从当前分类报告可以看到，不同类别在 precision、recall 与 F1-score 上并不均衡。对于分子亚型任务，宏平均指标比单纯 Accuracy 更能反映模型对低频类别的真实识别能力。当前任务中类别不平衡显著，导致少数类别对评估波动的贡献更大。

这类问题在生物医学分类中普遍存在。某些亚型天然低发，且部分亚型（如 Luminal A 与 Luminal B）分子边界重叠，导致模型在局部决策边界上更容易出现混淆。因此，单次实验的类级指标不宜被过度解读。

\subsection{特征数量对结果的影响}

高维特征筛选的一个重要作用是控制模型的有效复杂度。若保留的特征数过少，模型可能无法充分表达亚型差异；若特征数过多，则模型容易受到噪声干扰，尤其在支持样本有限的情况下，训练集表现会显著好于测试集。

当前“不做特征选择”的消融实验结果说明了这一点。理论上的“保留全部信息”在高维小样本条件下会转化为“保留全部噪声”，从而削弱判别边界。通过高方差筛选将模型注意力集中到稳定差异特征，有助于改善泛化。

\subsection{融合策略的敏感性}

Concat 和 MOFA 的比较也反映出多组学融合方法对参数和样本量的敏感性。Concat 的优势是确定性强；MOFA 引入潜在因子学习后，性能除受输入维度影响外，还受因子数、初始化和收敛行为影响。

因此，在当前实验规模下，Concat 更容易获得稳定基线，而 MOFA 的价值主要体现在结构提取与后续解释分析上。两者并非替代关系，而是可按任务目标互补使用。

\subsection{训练测试划分的敏感性}

训练-测试划分方式对本研究尤为关键。由于样本总数有限，每一次随机切分都会改变类别比例和难例分布。少数类上几个样本的变化即可显著影响最终指标。因此，单次实验适合做方法回归，不适合做最终结论。

从更严谨的角度，应采用多次重复划分、交叉验证或嵌套验证报告统计结果。当前代码结构已具备执行这类实验的基础，下一步重点是完善自动化统计输出与显著性检验流程。

\subsection{错误来源的分类}

对当前项目而言，错误来源可分为四类：
\begin{enumerate}
  \item 数据错误：样本名称或临床标签格式不一致；
  \item 预处理错误：训练/测试阶段特征顺序和统计量不一致；
  \item 接口错误：第三方库输入契约不满足；
  \item 评估错误：标签空间或概率列未对齐导致指标失真。
\end{enumerate}

本研究开发过程中已真实出现并修复了上述问题，包括：稀有类导致分层失败、预处理特征名不匹配、stacking 场景下非连续类标与概率列对齐问题。将这些工程错误显式写入论文，有助于提升方法复现成功率。

\subsection{结果稳定性的解释}

当前结果虽不足以形成大规模统计结论，但已呈现两个稳定趋势：其一，多组学融合链路是有效的；其二，特征筛选对性能与稳定性影响显著。

因此，本文更关注“模块作用可验证性”而非“单次最高分”。这也是将消融实验与误差分析放在正文核心位置的原因：它们直接回答方法链条是否必要、哪些环节最脆弱、如何提升复现可靠性。
