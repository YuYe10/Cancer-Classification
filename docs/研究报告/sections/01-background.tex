\section{背景与研究动机}

\subsection{乳腺癌分子异质性与分型的临床重要性}

乳腺癌作为全球高发恶性肿瘤，在诊断、治疗和预后评估中面临显著的生物学异质性挑战。虽然形态学分类（如组织学分级、TNM 分期）提供了基础的风险评估框架，但临床实践已充分证明，相同的病理分级往往预示着截然不同的临床结局。这一现象的根源在于肿瘤发生、演进与药物应答过程中存在深层的分子机制差异。

分子亚型的引入彻底改变了乳腺癌的临床管理范式。基于基因表达谱，Parker 等人（2009）首次定义了 PAM50 分子亚型系统，将乳腺癌分为 Luminal A、Luminal B、HER2-enriched 和 Basal-like 四个类别，这些亚型在激素受体表达状态、细胞增殖活跃程度、药物敏感性和长期生存期上存在显著区别。

\begin{itemize}
    \item \textbf{Luminal A}：激素受体阳性、HER2 阴性、增殖指数低，具有最优的预后
    \item \textbf{Luminal B}：激素受体阳性、增殖指数高或 HER2 阳性，中等预后
    \item \textbf{HER2-enriched}：HER2 过表达驱动，靶向治疗敏感性高
    \item \textbf{Basal-like}：激素受体阴性、增殖指数最高、预后最差
\end{itemize}

从治疗策略的角度看，分子亚型分类直接指导患者的个体化治疗方案：Luminal 亚型受益于内分泌治疗与 CDK4/6 抑制剂；HER2-enriched 亚型需要 HER2 靶向治疗（如曲妥珠单抗）；Basal-like 亚型通常对化疗敏感但内分泌治疗无效。因此，准确的分子亚型分类不仅具有科学价值，更直接关乎患者的治疗效果和生存预期。

\subsection{单一组学数据的表型解释局限性}

虽然基因表达数据在乳腺癌亚型分类中已建立起坚实的地位，但单一的转录组视角往往无法完整解释表型多样性。从生物学机制的深度来看，肿瘤的表型是多个调控层次的综合结果：

\begin{enumerate}
    \item \textbf{基因型层面}：DNA 序列变异、拷贝数变异和结构变异
    \item \textbf{表观遗传层面}：DNA 甲基化、组蛋白修饰等形成的异常调控
    \item \textbf{转录层面}：mRNA 和 lncRNA 的表达改变
    \item \textbf{蛋白层面}：翻译后修饰和蛋白定位
    \item \textbf{代谢层面}：关键代谢通路的重编程
    \item \textbf{微环境层面}：免疫细胞、基质细胞的相互作用
\end{enumerate}

仅使用 RNA-seq 数据进行分类具有以下局限：

\textbf{(a) 调控机制掩蔽}：某些基因可能由于启动子甲基化而被沉默，其 RNA 表达量反映的是最终结果，而非触发因素。两个表型相似的样本在 RNA 水平上表现接近，但甲基化驱动的调控状态可能差异巨大，这会影响长期的演进轨迹和对某些治疗的响应。

\textbf{(b) 噪声与可变性}：RNA 表达易受技术因素（如测序深度、批处理效应）和生物学因素（如细胞周期、微环境波动）影响。单一模态难以区分真正的表型差异与暂时的表达波动。

\textbf{(c) 亚型边界的模糊性}：传统基于表达阈值的分类往往存在连续性，而非离散的类别分界。相邻亚型样本在表达空间中相互接近，增加了分类错误率。

通过整合多组学信息，特别是将表达数据与甲基化调控信息联合，可以从多维度同时刻画肿瘤表型，实现更稳健、更准确的分类。

\subsection{多组学融合方法的发展轨迹与应用潜力}

多组学融合的发展经历了三个主要阶段：

\textbf{第一阶段：特征拼接（Feature Concatenation）}。最直观的融合方式是将不同模态的特征向量直接拼接，形成超高维特征空间。这种方法实现简单，但面临维度诅咒问题：在 $n \ll p$ 的环境下（样本数远小于特征数），模型极易过拟合。虽然如此，适当的特征选择后，拼接方法在乳腺癌分类中仍表现出竞争力。

\textbf{第二阶段：表示学习（Representation Learning）}。深度学习时代涌现了 Autoencoder、VAE、MOFA 等方法，它们试图先学习每个模态的低维表示，再进行融合。MOFA 是该阶段的代表性方法，它通过解析模式（Probabilistic Factor Model）建立可解释的潜因子结构，既能降低维度，又能对因子进行生物学解释。

\textbf{第三阶段：结构化学习（Structure-aware Learning）}。近期的方法倾向于显式引入生物网络结构，如蛋白质相互作用网络（PPI）、基因调控网络（GRN）等，通过图神经网络或路径分析获取更具有生物学意义的特征。

在本研究中，我们选择在前两阶段的框架内进行深入探索，通过对比拼接与 MOFA 两种策略的性能，评估表示学习在当前问题中的实际增益。这一选择既实用，又有助于揭示融合策略与任务特异性的关系。

\subsection{本研究的科学设计与核心问题}

本研究的设计围绕三个核心科学问题展开：

\textbf{核心问题 1：多组学融合是否能显著改进乳腺癌亚型分类的准确性与稳健性？}
我们的假设是通过整合表达与甲基化信息，模型能够从更多维度捕获亚型特异性的生物学特征，从而超越单一模态的性能基准。

\textbf{核心问题 2：不同融合策略（早期 vs. 晚期）在这一任务中的相对效能与适用条件如何？}
我们预期晚期融合（MOFA）虽然理论上能更好地学习潜在的共享与特有因子，但可能因样本规模和特征维度限制而不如简单的早期融合稳定，特别是在中等规模的真实医学数据集上。

\textbf{核心问题 3：特征筛选与预处理顺序在高维小样本分类中的关键程度？}
我们预期特征筛选与避免数据泄露的预处理顺序对分类性能的影响至关重要，其重要性甚至可能超过融合策略本身。

为了系统地回答这些问题，我们设计了以下实验体系：
\begin{itemize}
    \item \textbf{基准对比实验}：在相同的数据分割与评估方案下，对 RNA-only、Concat、MOFA 三种方案进行平衡的对标
    \item \textbf{消融实验体系}：通过去除甲基化、去除 RNA、关闭特征选择等操作，定量评估各模块的边际贡献与交互效应
    \item \textbf{稳健性验证}：采用交叉验证、随机种子检验等方法评估结果的可靠性与泛化能力
\end{itemize}

本文后续章节将围绕数据来源、特征处理、模型构建、实验设置和结果分析展开，力求在提供可复现框架的同时，通过严谨的方法设计回答上述科学问题。
