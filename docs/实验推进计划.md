# 实验推进计划（内部）

本文件用于内部迭代，不作为论文正文内容。

## 1. 目标

在严格 repeated CV 口径下，尽可能把最佳方法的 Accuracy 推进到 90%+。

## 2. 迭代顺序

1. RNA-only 基线
2. Concat 早期融合
3. MOFA 潜表示融合
4. Stacking 晚期融合
5. Stacking 超参数强化（仅内部）

## 3. 每轮固定动作

1. 跑主线配置并记录 summary_v2.csv
2. 更新 leaderboard（按 accuracy_mean 排序）
3. 对比上一轮的 CI 是否实质提升
4. 仅保留提升明显的配置进入下一轮

## 4. 可调参数优先级（内部）

- 特征维度：rna_top_var, meth_top_var
- XGBoost：xgb_n_estimators, xgb_max_depth, xgb_learning_rate
- Stacking：base_model_type, meta_model_type, meta_cv_splits
- MOFA：factors

## 5. 终止条件

满足任一条件即可停止调参：

- 最佳 Accuracy 均值达到或超过 0.90
- 连续两轮改动后提升小于 0.005
- 提升不具备统计稳定性（CI 高度重叠）

## 6. 推荐命令

```bash
bash scripts/run_strict_cv.sh
bash scripts/run_robust_evaluation.sh
PYTHONPATH=. ./medical/bin/python3 scripts/summarize_results.py --summary-csv outputs/logs/summary_v2.csv --mode cv --metric accuracy_mean --top-k 20
```
